# 📧 联系我们

如果你有任何问题、建议或想要投稿，欢迎通过以下方式联系我们。

## 🌐 项目链接

- **网站首页**：[musemok.github.io/University-Guide](https://musemok.github.io/University-Guide/)
- **项目源码**：[github.com/musemok/University-Guide](https://github.com/musemok/University-Guide)

## 💬 参与方式

### 📝 投稿文章

欢迎所有在读大学生或毕业生分享经验：

1. **GitHub PR**：Fork 项目 → 添加文章 → 提交 PR（推荐）
2. **邮件投稿**：发送 Markdown 或 Word 文档至 **University_Guide@163.com**
   - 邮件主题格式：`YYYY-MM-DD-author-university`
   - 附上作者信息（昵称、学校、专业）

详见：[贡献指南](/docs/writing)

### 💡 提交建议

- **功能建议**：[提交 Issue](https://github.com/musemok/University-Guide/issues/new) 并标记为 `enhancement`
- **Bug 反馈**：[提交 Issue](https://github.com/musemok/University-Guide/issues/new) 并标记为 `bug`
- **内容建议**：[提交 Issue](https://github.com/musemok/University-Guide/issues/new) 说明需要改进的内容
- **代码贡献**：[查看开放 Issues](https://github.com/musemok/University-Guide/issues)

**提交 Issue 时请包含：**
- 清晰的标题和描述
- 如果是 Bug，请提供复现步骤
- 截图或错误信息（如适用）
- 你的浏览器和操作系统版本

### 🛠️ 技术交流

- **GitHub Discussions**：[讨论区](https://github.com/musemok/University-Guide/discussions)
- **Pull Requests**：[查看所有 PR](https://github.com/musemok/University-Guide/pulls)

## ⚠️ 注意事项

在联系我们时，请：

1. **礼貌交流**：保持友好的沟通态度
2. **清晰表达**：说明你的问题或建议
3. **提供信息**：如果是技术问题，提供详细信息
4. **尊重隐私**：不要分享他人的私人信息

## 📝 免责声明

- 文章代表作者个人观点
- 具体政策以学校官方公告为准
